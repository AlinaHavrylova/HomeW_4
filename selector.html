Основні види селекторів

* Всі елементи
div * Всі дочірні елементи
tag По тегу
#id По id
.class По класу
tag1, tag2 Всі tag1 та tag2 елнменти
tag.class Всі елементи з конкретним класом
tag1 > tag2 Всі tag2, який є дочірний для tag1
tag1 + tag2 tag2, які знаходяться одразу після tag1
tag1 ~ tag2 Всі tag2, які знаходяться після tag2 (на тому самому рівні)
